关于 Hugo

Hugo 是一个用 Go 语言编写的快速、现代化的静态网站生成器。它非常适合用来搭建博客。

关于 GitHub Pages

GitHub Pages 是 GitHub 提供的静态网站托管服务，可以免费托管个人或组织的静态网站。
